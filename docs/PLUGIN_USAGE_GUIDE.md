# AI小说工坊 - 插件使用指南

## 插件系统概述

AI小说工坊插件系统提供了强大的扩展能力，允许用户根据需要安装和管理各种插件。

## 插件管理界面

### 访问插件管理

1. 打开应用程序
2. 进入项目编辑界面
3. 点击左侧菜单的"配置"
4. 在配置页面找到"插件管理"选项

### 界面说明

插件管理界面包含三个标签页：

#### 1. 已安装

显示所有已安装的插件列表，每个插件卡片包含：

- **插件图标和名称** - 快速识别插件
- **版本号和作者** - 插件基本信息
- **描述** - 插件功能说明
- **权限标签** - 插件所需的权限
- **状态开关** - 启用/停用插件
- **设置按钮** - 配置插件参数
- **卸载按钮** - 移除插件

#### 2. 扩展点

查看所有已注册的扩展点：

- **AI Provider** - 已注册的AI提供商
- **导出器** - 已注册的导出格式
- **导入器** - 已注册的导入格式
- **处理器** - 已注册的数据处理器
- **菜单项** - 已注册的菜单项
- **侧边栏面板** - 已注册的侧边栏面板
- **工具栏按钮** - 已注册的工具栏按钮

#### 3. 快捷命令

查看所有已注册的快捷命令，可以快速执行。

## 安装插件

### 方法一：从URL安装

1. 点击"安装插件"按钮
2. 选择"从URL安装"标签
3. 输入插件的manifest.json文件URL
4. 点击"安装"按钮

示例URL：
```
https://example.com/plugins/my-plugin/manifest.json
```

### 方法二：从本地文件安装

1. 点击"安装插件"按钮
2. 选择"从本地文件安装"标签
3. 拖拽或选择插件的JSON配置文件
4. 等待安装完成

### 方法三：从插件市场安装（即将推出）

1. 点击"安装插件"按钮
2. 选择"插件市场"标签
3. 浏览或搜索可用插件
4. 点击"安装"按钮

## 插件配置

### 打开插件设置

1. 在已安装插件列表中找到目标插件
2. 确保插件已启用（状态开关打开）
3. 点击"设置"按钮

### 配置项说明

不同插件有不同的配置项，常见类型包括：

- **文本输入框** - 输入API密钥、URL等
- **数字输入框** - 设置数值参数
- **开关** - 启用/禁用功能
- **下拉选择** - 选择预设选项
- **多选框** - 选择多个选项

### 保存配置

配置完成后点击"保存"按钮，配置会立即生效。

## 插件状态管理

### 启用插件

1. 找到目标插件
2. 将状态开关切换到"已启用"
3. 等待激活完成

### 停用插件

1. 找到目标插件
2. 将状态开关切换到"已停用"
3. 等待停用完成

### 卸载插件

1. 找到目标插件
2. 点击"卸载"按钮
3. 确认卸载操作
4. 等待卸载完成

⚠️ **注意**：卸载插件会删除所有相关数据和配置。

## 内置插件示例

### 1. EPUB导出器

**功能**：将小说导出为EPUB格式

**权限**：文件系统访问

**使用方法**：
1. 安装并启用插件
2. 在章节管理页面点击"导出"
3. 选择"EPUB"格式
4. 配置导出选项
5. 开始导出

### 2. PDF导出器

**功能**：将小说导出为PDF格式

**权限**：文件系统访问

**配置项**：
- 页面大小：A4、A5、Letter、Legal
- 字号：正文默认字号
- 行高：行间距倍数
- 页边距：上下左右边距
- 包含目录：是否生成目录
- 包含封面：是否生成封面

### 3. 文本清洗处理器

**功能**：在导入文本时自动清洗格式

**处理阶段**：导入前（pre-import）

**功能包括**：
- 移除BOM标记
- 统一换行符
- 移除行尾空格
- 压缩多余空行
- 移除特殊字符
- 统一标点符号
- 修正引号配对

### 4. 智谱GLM Provider

**功能**：提供智谱GLM系列模型支持

**权限**：AI API访问、网络请求

**配置项**：
- API密钥：智谱GLM的API密钥
- 默认模型：GLM-4、GLM-4-Air、GLM-4-Flash

**使用方法**：
1. 获取智谱GLM API密钥
2. 安装并启用插件
3. 在设置中输入API密钥
4. 选择默认模型
5. 在AI助手设置中选择"智谱GLM"

## 常见问题

### Q: 插件安装后没有显示？

A: 检查以下几点：
1. 插件是否成功安装（查看控制台日志）
2. 插件是否已启用
3. 插件是否有错误（查看浏览器控制台）
4. 刷新页面后重试

### Q: 插件配置不生效？

A: 尝试以下步骤：
1. 停用插件再重新启用
2. 清除浏览器缓存
3. 检查配置格式是否正确
4. 查看浏览器控制台的错误信息

### Q: 如何更新插件？

A: 目前需要：
1. 卸载旧版本插件
2. 安装新版本插件
3. 重新配置插件设置

### Q: 插件会影响性能吗？

A:
- 已停用的插件不会影响性能
- 启用的插件根据功能不同，影响程度不同
- 建议只启用需要的插件
- 定期清理不用的插件

### Q: 插件数据存储在哪里？

A:
- 插件配置：localStorage
- 插件状态：Pinia store
- 插件代码：内存中（需要重新安装）

### Q: 如何备份插件配置？

A: 使用插件导出功能（即将推出）：
1. 进入插件设置
2. 点击"导出配置"
3. 保存JSON文件

## 安全建议

1. **只安装可信插件** - 从官方渠道或可信来源安装
2. **检查权限** - 安装前查看插件所需的权限
3. **定期审查** - 定期检查已安装的插件
4. **及时更新** - 保持插件为最新版本
5. **报告问题** - 发现问题及时反馈

## 性能优化

1. **按需启用** - 只启用必要的插件
2. **清理缓存** - 定期清理插件缓存
3. **卸载不用** - 卸载不再使用的插件
4. **监控资源** - 注意插件对系统资源的使用
