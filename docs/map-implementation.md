## 技术选型

### Konva.js 方案优势
1. **性能优异**：基于 Canvas 渲染，支持大量元素（100+地点）
2. **交互丰富**：内置拖拽、缩放、事件系统
3. **Vue 集成**：vue-konva 提供良好的 Vue 3 支持
4. **轻量灵活**：相比 Leaflet（地图库）更适合自定义地图
5. **导出方便**：原生支持导出为图片

### 与其他方案对比

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| Konva.js | 性能高、交互强、自定义能力强 | 需手动实现地图功能 | ✅ 本项目 |
| Leaflet | 专业地图库、功能完善 | 需要地理坐标、不够灵活 | 真实地理地图 |
| SVG + D3 | 矢量图形、动画强大 | 性能较差、复杂度高 | 数据可视化 |

## 数据流程

```
WorldSetting (世界观设定)
    ↓
extractMapDataFromWorld() (提取数据)
    ↓
MapData (地图数据)
    ↓
WorldMap.vue (渲染 + 编辑)
    ↓
saveMapData() (保存)
    ↓
WorldSetting.geography.locations (更新)
```

## 性能优化

1. **视口裁剪**：只渲染可见区域内的元素
2. **图层分离**：独立控制各图层显示
3. **事件防抖**：缩放、拖拽等事件优化
4. **按需渲染**：图层隐藏时不渲染
5. **Canvas 缓存**：静态元素缓存提升性能

## 使用示例

### 添加地点
1. 点击工具栏"添加地点"按钮
2. 或选择"添加地点"工具后点击地图
3. 在弹窗中填写地点信息
4. 点击"保存"

### 编辑地点
1. 双击地图上的地点图标
2. 或点击侧边栏地点列表项
3. 在弹窗中修改信息
4. 点击"保存"

### 添加势力范围
1. 切换到"区域"标签
2. 点击"添加区域"
3. 设置区域名称、颜色、势力关联
4. 编辑区域顶点坐标
5. 点击"保存"

### 导出地图
1. 点击"导出"按钮
2. 选择导出格式（JSON/SVG/PNG/CSV）
3. 根据需要设置选项
4. 点击"导出"下载文件
