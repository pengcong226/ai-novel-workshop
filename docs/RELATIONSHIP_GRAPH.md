# 人物关系图功能说明

## 功能概述

人物关系图可视化功能提供了一个交互式的网络图，用于展示和分析小说中人物之间的关系。

## 主要功能

### 1. 可视化展示

- **节点显示**：每个人物以节点形式展示
  - 节点颜色：根据性别区分（蓝色=男，红色=女，灰色=其他）
  - 节点大小：根据出场次数动态调整
  - 节点边框：主要角色有金色边框

- **边显示**：关系以连线形式展示
  - 线条颜色：根据关系类型区分
    - 家人：绿色
    - 朋友：蓝色
    - 敌人：红色
    - 恋人：橙色
    - 对手：灰色
    - 其他：浅灰色
  - 线条粗细：根据关系强度调整
  - 标签：显示关系类型

### 2. 交互功能

- **拖拽节点**：可以拖动人物节点调整位置
- **缩放画布**：支持鼠标滚轮缩放
- **拖拽画布**：可以拖动整个画布
- **悬停提示**：鼠标悬停在节点上显示详细信息
- **点击节点**：查看人物详情和相关人物
- **右键节点**：高亮显示相关人物节点
- **右键边**：编辑关系属性

### 3. 过滤功能

- **关系类型过滤**：选择要显示的关系类型
- **关系强度过滤**：设置最小关系强度
- **隐式关系开关**：显示/隐藏从章节互动中提取的关系
- **人物搜索**：聚焦到特定人物及其关系网络

### 4. 关系提取

#### 显式关系
从 `Characters.vue` 中的人物设定提取 `relationships` 字段。

#### 隐式关系
从章节内容中分析人物互动：
- 统计同一章节中出现的人物
- 至少3次共同出场视为存在互动关系
- 自动添加"互动"类型的关系

### 5. 导出功能

支持将关系图导出为PNG图片。

## 使用方法

### 访问关系图

1. 打开项目编辑器
2. 在左侧菜单中点击"人物关系图"

### 添加关系

在人物设定页面中：
1. 编辑人物
2. 切换到"人际关系"标签
3. 添加关系对象和类型
4. 保存人物

### 编辑关系

在关系图中：
1. 右键点击关系连线
2. 修改关系类型和描述
3. 保存更改

### 查看人物详情

点击人物节点，可以查看：
- 基本信息（性别、年龄）
- 性格特点
- 能力列表
- 出场次数
- 相关人物列表

### 高亮关系网络

右键点击人物节点，可以：
- 高亮显示该人物的所有关系
- 突出显示相关人物节点

### 导出图片

点击右上角"导出图片"按钮，保存当前关系图为PNG格式。

## 技术实现

### 核心文件

1. **RelationshipGraph.vue** - 关系图组件
   - 路径：`src/components/RelationshipGraph.vue`
   - 功能：图可视化界面和交互

2. **relationExtractor.ts** - 关系提取工具
   - 路径：`src/utils/relationExtractor.ts`
   - 功能：从人物和章节提取关系数据

3. **ProjectEditor.vue** - 项目编辑器（已更新）
   - 添加了"人物关系图"菜单项
   - 集成了 RelationshipGraph 组件

### 使用的技术

- **AntV G6**：图可视化引擎
  - 力导向布局算法
  - 丰富的交互支持
  - 高性能渲染

- **Vue 3 Composition API**：组件架构
- **Element Plus**：UI组件库
- **TypeScript**：类型安全

### 数据流

```
Characters (人物设定)
    ↓
extractRelationships() (提取显式关系)
    ↓
Chapters (章节内容)
    ↓
extractFromChapters() (提取隐式关系)
    ↓
mergeRelationships() (合并关系)
    ↓
filterGraphData() (应用过滤器)
    ↓
G6 Graph (可视化展示)
```

## 性能优化

### 大量人物处理

当人物数量较多时（>100），建议：

1. 使用人物过滤器，只显示关注的人物
2. 设置关系强度过滤，隐藏弱关系
3. 关闭隐式关系显示

### 布局优化

使用力导向布局算法，自动避免节点重叠。

参数配置：
- `preventOverlap: true` - 防止节点重叠
- `linkDistance: 150` - 边的理想长度
- `nodeStrength: -30` - 节点间斥力
- `edgeStrength: 0.1` - 边的引力

## 常见问题

### Q: 为什么某些人物没有出现在图中？

A: 可能原因：
1. 该人物没有设置任何关系
2. 被过滤器过滤掉了
3. 该人物未在任何章节出场（如果开启了隐式关系过滤）

解决方案：
- 检查人物设定中的关系
- 调整过滤器设置
- 关闭"隐式关系"过滤

### Q: 如何删除关系？

A: 目前需要：
1. 进入人物设定页面
2. 编辑对应人物
3. 在"人际关系"标签中删除关系
4. 保存人物

未来会添加在关系图中直接删除的功能。

### Q: 关系强度是如何计算的？

A: 关系强度根据类型预设：
- 家人、敌人、恋人：强度5（最强）
- 朋友：强度4
- 对手：强度3
- 其他：强度2

隐式关系根据共同出场次数计算，最多为5。

### Q: 导出的图片分辨率如何？

A: 导出图片分辨率与画布显示尺寸一致。建议：
1. 放大画布到合适大小
2. 调整布局
3. 再导出图片

未来会支持自定义分辨率导出。
