# 人物关系图 - 快速开始指南

## 功能概览

人物关系图功能已成功集成到 AI 小说工坊中，提供交互式的网络图可视化，帮助您：

- 📊 可视化人物关系网络
- 🔍 分析人物互动模式
- ✏️ 编辑和管理工作关系
- 📸 导出关系图用于参考

## 快速开始

### 1. 访问关系图

启动开发服务器后：
```bash
npm run dev
```

在浏览器中打开 http://localhost:3001，然后：
1. 创建或打开一个项目
2. 在左侧菜单中点击"人物关系图"

### 2. 添加人物关系

在"人物设定"页面中：
1. 创建人物（或使用AI生成）
2. 编辑人物信息
3. 切换到"人际关系"标签
4. 添加关系：
   - 选择关系对象（其他人物）
   - 选择关系类型（家人、朋友、敌人、恋人、对手、其他）
   - 添加关系描述
5. 保存人物

### 3. 查看关系图

返回"人物关系图"页面，您将看到：
- **节点**：代表人物
  - 蓝色节点 = 男性角色
  - 红色节点 = 女性角色
  - 灰色节点 = 其他
  - 节点越大 = 出场次数越多
  - 金色边框 = 主要角色

- **连线**：代表关系
  - 绿色 = 家人
  - 蓝色 = 朋友
  - 红色 = 敌人
  - 橙色 = 恋人
  - 灰色 = 对手
  - 线条越粗 = 关系越强

### 4. 交互操作

- **拖拽节点**：调整人物位置
- **缩放**：鼠标滚轮
- **平移画布**：拖拽空白区域
- **查看详情**：点击节点
- **高亮关系网络**：右键点击节点
- **编辑关系**：右键点击连线

### 5. 过滤和搜索

使用顶部过滤器：
- **关系类型**：选择要显示的关系类型
- **最小强度**：过滤弱关系
- **显示隐式关系**：从章节互动中提取的关系
- **搜索人物**：聚焦到特定人物

### 6. 导出图片

点击右上角"导出图片"按钮，保存为PNG格式。

## 隐式关系发现

当您的章节内容中有多个角色共同出现时，系统会自动识别：

1. 在章节中写人物互动
2. 多次共同出现（≥3次）会被识别为"互动"关系
3. 在关系图中显示为浅灰色连线

例如：
```
第1章：李明、林月、张伟一起探险...
第3章：李明和林月训练...
第5章：林月救助李明...
```

系统会自动识别李明-林月的互动关系。

## 性能建议

### 大量人物（>50）

建议使用过滤器：
```javascript
// 只显示特定人物
selectedCharacters: ['character-id-1', 'character-id-2']

// 只显示强关系
minStrength: 4

// 隐藏隐式关系
showImplicit: false
```

### 大量关系（>200）

考虑分批显示：
1. 按关系类型分批查看
2. 聚焦主要角色
3. 过滤弱关系

## 故障排除

### 问题：关系图不显示

**检查清单**：
- [ ] 是否已创建人物？
- [ ] 人物是否设置了关系？
- [ ] 浏览器控制台是否有错误？
- [ ] 开发服务器是否正常运行？

### 问题：节点重叠

**解决方案**：
- 拖拽节点调整位置
- 刷新页面重新布局
- 放大画布查看

### 问题：导出图片模糊

**解决方案**：
1. 放大画布到合适大小
2. 调整布局
3. 再导出图片
