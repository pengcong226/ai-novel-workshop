# Novel Editor Features Backlog

这是记录我们针对“缺口B：正文内联AI编辑器 (Inline AI Editor)”进行脑暴的待开发功能清单。
等后续项目进度推进到这里时，可以随时取用这些点子。

## 1. 幽灵导演模式 (Ghost Director Mode)
- **目标：** 解决用户“只有大纲骨架，不知如何填充血肉展开描写”的痛点。
- **方案描述：**
  - 作者在正文编辑器内，直接使用特殊占位符（如 `[导演：这里加一段环境描写，突出阴森，50字]`）来写提示词。
  - 按下快捷键后，AI 直接接管编辑器，将这些带有标记的标签原地“膨胀”成具体的文段。
  - 保留用户的骨干控制力，同时大幅度自动化补充文笔。

## 2. 时空批注/查漏补缺系统 (Chronicle Margin-Chat)
- **目标：** 解决作者“写到一半怕吃书、怕出现逻辑漏洞”的顾虑，提供实时纠错副驾驶。
- **方案描述：**
  - 不打断正文内容。作者可以选中一段存疑的文本，在右侧边缘空白处（Margin）弹出一个便利贴对话框。
  - 作者提问：“主角现在杀这个人会引起什么后果？”
  - AI 根据沙盒底层的记忆和图谱分析这段话，并在便利贴里给出评估报告以及“建议改写的版本”。
  - 用户满意后可一键 `[应用替换]` 正文。

## 3. 悬浮指令菜单 (Hover Toolbar)
- **目标：** 快速修改已有的文字段落（类似 Notion AI）。
- **方案描述：**
  - 选中一段文字，上方悬浮菜单提供：【改写风格】、【扩写】、【缩写】。
  - 提供【一键提取到图谱】功能，AI自动把这段剧情里出现的新人物或新关系提取到沙盒数据库中。

## 4. 沉浸式行内百科热链接 (Inline Lore Links)
- **目标：** 防止写作时遗忘设定。
- **方案描述：**
  - 像 Obsidian 一样打出 `[[` 自动补全当前世界观里的实体（Entity）。
  - 悬停在人物名字上时，原地弹出小卡片，显示他的身份、境界以及与主角的好感度/态度。
