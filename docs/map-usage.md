# 世界地图可视化功能使用说明

## 功能概述

世界地图可视化功能允许您创建交互式地图来展示故事世界中的地点、区域和路线。

## 主要特性

### 1. 地点管理
- **添加地点**：点击工具栏的"添加地点"按钮或使用快捷键
- **编辑地点**：双击地图上的地点图标进行编辑
- **拖拽移动**：选中地点后可直接拖拽调整位置
- **地点类型**：支持城市、城镇、村庄、山脉、河流、湖泊、森林、沙漠、海洋、岛屿、遗迹、地下城、城堡、神庙等
- **重要性分级**：高、中、低三个级别，影响图标大小

### 2. 区域划分
- 创建势力范围、国家边界等区域
- 自定义区域颜色和透明度
- 支持多边形区域绘制

### 3. 路线系统
- 添加道路、小径、河流、边界等路线
- 自定义路线颜色和宽度
- 支持多点路径

### 4. 交互功能
- **缩放**：鼠标滚轮或工具栏按钮
- **平移**：选择"平移"工具后拖动地图
- **图层控制**：显示/隐藏地点、区域、路线、人物位置

### 5. 导出功能
- **JSON**：导出完整地图数据，方便备份和迁移
- **SVG**：矢量图格式，适合打印和编辑
- **PNG**：位图格式，方便分享
- **CSV**：仅导出地点数据

## 使用流程

### 第一步：自动提取地点
当您首次打开地图页面时，系统会自动从世界观设定中提取地点信息，并为每个地点分配随机位置。

### 第二步：调整布局
- 使用"自动布局"功能让系统自动排列地点
- 或手动拖拽地点到合适位置

### 第三步：编辑地点
- 双击地点图标打开编辑对话框
- 设置地点名称、类型、描述、重要性、颜色等属性
- 关联所属势力

### 第四步：添加区域和路线
- 根据需要添加势力范围、边界等区域
- 添加道路、河流等路线连接地点

### 第五步：保存和导出
- 点击"保存地图"将地图数据保存到项目
- 使用"导出"功能导出为图片或数据文件

## 常见问题

**Q: 如何添加地形？**
A: 目前可以通过添加地点并设置类型（山脉、森林等）来表示地形。未来会添加地形绘制工具。

**Q: 可以导入自定义地图背景吗？**
A: 目前支持设置背景颜色。未来版本将支持导入图片作为地图背景。

**Q: 地图数据会自动保存吗？**
A: 不会自动保存，需要点击"保存地图"按钮手动保存。建议定期保存避免数据丢失。

**Q: 如何显示人物位置？**
A: 在世界观设定中关联人物和地点，然后在地图页面勾选"人物位置"图层。
