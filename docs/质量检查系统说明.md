# 质量检查系统 - 实现说明

## 功能概述

已成功实现多维度质量评估和改进建议系统，包括以下核心功能：

## 1. 质量检查核心 (qualityChecker.ts)

**位置**: `src/utils/qualityChecker.ts`

### 五大质量维度

1. **情节质量** (Plot Quality)
   - 冲突设置检测
   - 节奏变化分析
   - 情节推进流畅性
   - 高潮点识别

2. **人物塑造** (Character Quality)
   - 人物出场识别
   - 对话自然度检测
   - 性格特征展现
   - 人物互动分析

3. **文笔水平** (Writing Quality)
   - 句子流畅度
   - 描写丰富度
   - 对话自然度
   - 重复用词检测
   - 修辞手法识别

4. **逻辑一致性** (Logic Consistency)
   - 世界观设定一致性
   - 时间线一致性
   - 人物状态一致性
   - 因果关系检测

5. **创新性** (Innovation)
   - 情节套路识别
   - 人物设定创新性
   - 世界观创新性
   - 情节反转检测

### 主要类和方法

```typescript
class QualityChecker {
  // 检查单个章节
  async checkChapter(chapter: Chapter): Promise<QualityReport>

  // 批量检查章节
  async checkChapters(chapters: Chapter[]): Promise<QualityReport[]>
}

// 创建质量检查器
function createQualityChecker(
  worldSetting?: WorldSetting,
  characters?: Character[],
  outline?: Outline,
  config?: Partial<QualityCheckConfig>
): QualityChecker

// 分析质量趋势
function analyzeQualityTrend(reports: QualityReport[]): TrendAnalysis
```

## 2. 质量报告界面 (QualityReport.vue)

**位置**: `src/components/QualityReport.vue`

### 界面功能

- **总体概览**: 显示平均分、质量趋势、待改进章节数
- **质量趋势图表**: 使用 ECharts 绘制趋势折线图
- **维度分析雷达图**: 直观展示各维度得分
- **章节详情列表**: 支持搜索和筛选
- **改进建议展示**: 提供具体改进方向

### 图表展示

- **趋势图**: 展示质量分数随章节变化的趋势
- **雷达图**: 展示各维度的平均、最佳、最差表现
- **维度卡片**: 每个维度的独立展示

## 3. 章节管理集成 (Chapters.vue)

**修改**: `src/components/Chapters.vue`

### 新增功能

- **单章质量检查**: 在章节编辑器中点击"质量检查"按钮
- **质量评分显示**: 章节列表显示质量评分
- **详细报告对话框**: 显示维度评分、问题详情、改进建议

### 使用方式

1. 编辑章节时点击"质量检查"按钮
2. 系统自动分析并生成报告
3. 在对话框中查看详细结果
4. 根据建议进行修改

## 4. 报告导出工具 (reportExporter.ts)

**位置**: `src/utils/reportExporter.ts`

### 导出格式

1. **JSON 格式**
   - 完整的结构化数据
   - 包含趋势分析
   - 便于程序处理

2. **Markdown 格式**
   - 易读的文本报告
   - 包含总体统计和章节详情
   - 适合文档归档

3. **打印格式**
   - 直接打印或保存为 PDF
   - 格式化的 HTML
   - 包含样式和图表

### 导出方法

```typescript

// 导出为 Markdown

// 打印报告
printQualityReport(reports, projectName)
```

## 6. 可配置性

### 检查配置

```typescript
interface QualityCheckConfig {
  enablePlotCheck: boolean        // 启用情节检查
  enableCharacterCheck: boolean   // 启用人物检查
  enableWritingCheck: boolean     // 启用文笔检查
  enableLogicCheck: boolean       // 启用逻辑检查
  enableInnovationCheck: boolean  // 启用创新性检查
  customRules: CustomRule[]       // 自定义规则
  qualityThreshold: number        // 质量阈值
}
```

### 自定义规则

```typescript
interface CustomRule {
  id: string
  name: string
  description: string
  pattern: string              // 正则表达式或关键词
  type: 'regex' | 'keyword' | 'custom'
  severity: number            // 严重程度 1-10
  enabled: boolean
}
```

## 8. 使用示例

### 单章节检查

```typescript
import { createQualityChecker } from '@/utils/qualityChecker'

const checker = createQualityChecker(
  project.world,
  project.characters,
  project.outline,
  project.config
)

const report = await checker.checkChapter(chapter)
console.log('质量评分:', report.overallScore)
console.log('改进建议:', report.improvements)
```

### 批量检查

```typescript
const reports = await checker.checkChapters(chapters, (current, total, chapterNumber) => {
  console.log(`正在检查第 ${chapterNumber} 章 (${current}/${total})`)
})

const trend = analyzeQualityTrend(reports)
console.log('平均分:', trend.averageScore)
console.log('趋势:', trend.scoreTrend)
```
