## 3. 核心优势与亮点

### 3.1 架构亮点

#### 三层记忆系统
```
短期记忆 → 最近3-5章原文 + 当前章节上下文
中期记忆 → 章节摘要 + 人物状态快照
长期记忆 → 向量化存储 + 语义检索
```
这是项目的核心创新点，解决了长篇小说创作中的上下文一致性问题。

#### 分层AI模型策略
| 任务类型 | 模型选择 | 成本占比 |
|---------|---------|---------|
| 规划任务 | GPT-4/Claude | 10% |
| 写作任务 | GPT-3.5/本地模型 | 80% |
| 检查任务 | GPT-3.5/本地模型 | 10% |

相比全程使用GPT-4，可节省约89%成本。

### 3.2 技术创新

1. **智能上下文注入**：根据当前内容自动检索相关设定
2. **成本优化策略**：差异化模型使用 + 预算控制
3. **本地优先**：数据完全本地存储，保护用户隐私
4. **流式响应**：支持实时生成显示，提升用户体验

### 3.3 用户体验设计

- **三栏布局**：左侧导航 + 中间编辑 + 右侧AI助手
- **实时保存**：自动保存到IndexedDB，防止数据丢失
- **项目管理**：支持导出/导入 `.anproj` 格式
- **配置灵活**：支持多种AI提供商、预设模板、高级参数调整
